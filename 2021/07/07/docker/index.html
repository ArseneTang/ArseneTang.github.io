【启动docker】service docker start

【搜索相关镜像】docker search

【下载某个镜像】docker pull

【删除某个镜像】 docker rmi —(注意删除时候要停掉并删除使用该镜像的docker,包括没有成功启动的)

【分配一个IP】docker run -d -p :80

【得到docker相关数据】docker ps

【查看已有镜像】docker image ls

【启动某个docker】 docker start [container id]

【停止某个docker】docker stop [container id]

【复制宿主机的文件到docker】docker cp <宿主机文件地址> [container id]:<docker目标地址>

【复制docker的文件到宿主机】docker container cp [container id]:<docker文件地址> <宿主机目标地址>

【进入docker内部】docker exec -it [container id] bash

【删除docker内部某个文件】rm -rf

【删除某个docker】docker rm [container id] (需要先停止再删除)

【删除所有docker】docker rm $(docker ps -a | awk ‘{ print $1}’ | tail -n +2)

【删除所有镜像】docker rmi $(docker images | awk ‘{print $3}’ |tail -n +2)

【删除所有没有运行的容器】docker container prune

【启动所有容器】docker start $(docker ps -a | awk ‘{ print $1}’ | tail -n +2)

【关闭所有容器】docker stop $(docker ps -a | awk ‘{ print $1}’ | tail -n +2)

【退出docker容器内部】exit
